Event Registration API built with Fastify, Drizzle ORM, and PostgreSQL. Features participant sign-up, invite tracking, and referral-based ranking system.
⚠️ In development - Final improvements in progress.

## 🛠️ Technologies Used

| Tech         | Description                                      |
|--------------|--------------------------------------------------|
| **Fastify**  | Fast and efficient Node.js web framework         |
| **Drizzle ORM** | Lightweight and type-safe ORM for TypeScript |
| **PostgreSQL** | Relational database.     |
| **Redis**    | In-memory cache for performance optimization     |
| **Jest**     | Testing framework for unit/integration tests     |
| **TypeScript** | Strongly typed language for safer code         |
